# Loan Approval Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on loan approval data to identify factors affecting loan approvals and provide business insights.

## Dataset
- 367 loan applications
- Features include applicant income, co-applicant income, loan amount, credit history, education, and property area.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning
- Missing Value Treatment
- Statistical Analysis
- Univariate Analysis
- Bivariate Analysis
- Correlation Heatmap
- Business Recommendations

## Key Findings
- Applicant income positively correlates with loan amount.
- Urban applicants dominate the dataset.
- Most applicants have good credit history.
- Standard loan term is 360 months.
